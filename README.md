# 📚 GenAI Student Assistant

Upload a short PDF (1–5 pages) and choose:

- ✨ Summarize
- 📝 Generate Quiz
- 🌐 Translate to Tamil

This app uses:
- 🤖 `flan-t5` for summarizing & quiz
- 🇮🇳 `suriya7/English-to-Tamil` for Tamil translation
- ⚡ Gradio for the UI
